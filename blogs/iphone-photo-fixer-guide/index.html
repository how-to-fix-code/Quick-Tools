What is a HEIC file?

HEIC (High Efficiency Image Container) is the modern file format Apple has used since iOS 11. As the name suggests, it is highly efficient. It can store high-quality images at roughly half the file size of a standard JPEG.

This is great for saving space on your iPhone, but it's a headache for compatibility. Windows, Android, and many older web browsers don't natively support HEIC, meaning you can't view these photos without installing paid extensions or using a converter.
